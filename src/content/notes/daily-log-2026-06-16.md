- 系统学习 AI Agent 的底层机制：
  - 普通 LLM 调用 vs Agent。
  - OpenAI-compatible API。
  - Tool Calling / Function Calling。
  - Agent Loop。
  - Memory / Session。
  - Trace / Metrics / Runtime。

### 有价值的收获

- Agent = LLM + Tool + Loop。
- LLM 负责决策，Python 负责执行；模型不会执行工具，只会提出 tool_call。
- Memory 本质是 messages，不是魔法。
- Agent 的灵魂不是工具数量，而是可观察、可停止、可追踪的运行循环。
- 已从“会调用 LLM API”推进到“能手写 Agent Runtime V1”。
