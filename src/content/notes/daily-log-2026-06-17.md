- 完成并 AC `503. Next Greater Element II`。
- 核心理解：用 `i % numsSize` 模拟环形数组，遍历 `2 * numsSize`，但只在第一轮入栈，第二轮只负责补答案。
- 继续推进 `42. Trapping Rain Water`。

- 最后切回标准单调栈：当前柱子作为右边界，弹出元素作为低洼点，新栈顶作为左边界，用 `width * boundedHeight` 结算一层水。

### Java / JVM

- `HashMap`：数组 + 链表 + 红黑树、Java 8 树化、2 的幂容量、负载因子 0.75。
- `ArrayList` vs `LinkedList`：动态数组与双向链表的差异、随机访问、插入删除、实际开发选择。
- JVM 运行时内存区域：堆、虚拟机栈、本地方法栈、方法区、程序计数器。
- GC Roots 与可达性分析。
- 类加载过程与双亲委派机制。

### Spring Boot / SSM / 数据库

- `@SpringBootApplication`：`@SpringBootConfiguration`、`@EnableAutoConfiguration`、`@ComponentScan`。
- Spring Boot 自动配置：`AutoConfigurationImportSelector`、条件装配、用户 Bean 优先。
- Spring MVC 请求流程：`DispatcherServlet -> HandlerMapping -> HandlerAdapter -> Controller -> Service -> Mapper -> JSON/View`。
- MyBatis：`#{}` vs `${}`，预编译占位符 vs 字符串拼接，SQL 注入风险。
- `@Transactional` 失效场景：内部调用、异常被捕获、默认不回滚受检异常、类或方法没有被代理。
- 数据库索引：B+Tree、回表、覆盖索引、最左前缀、索引失效场景。

### WebSocket / 聊天室项目表达

- HTTP 是请求-响应模型，WebSocket 是全双工长连接。
- WebSocket 建连：TCP 三次握手 -> 客户端 HTTP Upgrade -> 服务端返回 `101 Switching Protocols` -> 升级为 WebSocket。
- 聊天室连接管理：`userId -> session`、`roomId/channelId -> sessions`。
- 消息流程：鉴权 -> 入库 -> 广播。
- 心跳机制：ping / pong 或业务心跳，检测异常断开并释放无效连接。
- 多实例扩展：Nginx 负载均衡、Redis 维护在线状态和房间映射、RabbitMQ/MQ 做跨实例消息广播和削峰。

### Vue3 / 前端三件套

- Vue2 vs Vue3 响应式：`Object.defineProperty` vs `Proxy`。
- `ref` vs `reactive`：基本类型 / 对象状态、`.value`。
- 组件通信：props、emit、provide/inject、Pinia。
- `v-if` vs `v-show`：DOM 创建销毁 vs `display` 显隐。
- JS 事件循环：同步代码 -> 微任务 -> 宏任务。
- CSS 盒模型：content、padding、border、margin，标准盒模型和 `border-box`。

### AI / 大模型框架

- 普通 LLM 应用：API + Prompt + Context + 输出格式。
- Agent 应用：LLM + Tool Calling + Loop + Memory + Runtime。
- Tool Calling：模型不是真的执行工具，而是输出结构化工具调用请求，Runtime 真正执行工具。
- RAG：检索外部资料再生成，不改模型参数。
- LangChain / LangGraph / Dify：代码工具箱、Agent 流程图、可视化 AI 应用平台。
- 企业知识库问答系统：文档切分、Embedding、向量数据库、相似度检索、Prompt 组装、LLM 基于资料回答。

### 微信小程序 / J2EE 兜底

- 小程序文件结构：`wxml`、`wxss`、`js`、`json`。
- 生命周期：`onLoad`、`onShow`、`onReady`、`onHide`、`onUnload`。
- 网络请求：`wx.request`。
- 视图更新：`this.setData`。
- Servlet：Java Web 底层请求处理规范。
- Tomcat：Servlet 容器。
- `DispatcherServlet`：Spring MVC 的核心 Servlet。
